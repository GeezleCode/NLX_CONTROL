spk_findAnalogEventBin

PURPOSE

returns analog data for a given event

SYNOPSIS

function [v,t,i,terr] = spk_findAnalogEventBin(s,ChanName,EventLabel)

DESCRIPTION 

 returns analog data for a given event
 uses s.currenttrials. 
 v ...... [nChans x nTrials] analog value 
 t ...... [nChans x nTrials] event time 
 i ...... [nChans x nTrials] bin number  
 terr ... [nChans x nTrials] time error of bin in reference to event

CROSS-REFERENCE INFORMATION

This function calls:
  • spk_AnalogTimeVec retrieves times of analog data bins
  • spk_findAnalog finds an analog channel, or channels featuring a pattern
This function is called by:
  • spk_AnalogRemoveLineWin fits a sine wave of frequency f to each trial and substract the mean
