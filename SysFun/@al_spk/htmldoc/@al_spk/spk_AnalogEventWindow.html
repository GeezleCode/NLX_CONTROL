spk_AnalogEventWindow

PURPOSE

returns start and end bins of a window defined by an event(s)

SYNOPSIS

function [b,t,tVec] = spk_AnalogEventWindow(s,Ev,EvOffset)

DESCRIPTION 

 returns start and end bins of a window defined by an event(s)
 [b,t] = spk_AnalogEventWindow(s,Ev,EvOffset)

 Ev ......... eventlabel
 EvOffset ... eventlabel or a [nx2] matrix defining an offset to Ev

 b ...... window as bin nr
 t ...... window as time
 tVec ... sample times of each analog channel

CROSS-REFERENCE INFORMATION

This function calls:
  • spk_AnalogTimeVec retrieves times of analog data bins
  • spk_TrialNum returns and check for number of trials in @al_spk object
  • spk_getEventWindow returns start and end time of a window defined by an event(s)
This function is called by:
  • spk_AnalogIsSaturated detects trials exceeding a range of analog values
  • spk_AnalogRemoveLine fits a sine wave of frequency f to each trial and substract the mean
  • spk_getAnalog extracts analog data from the object.
