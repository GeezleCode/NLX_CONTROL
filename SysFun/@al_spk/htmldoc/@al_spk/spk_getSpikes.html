spk_getSpikes

PURPOSE

get the spikes within a time window

SYNOPSIS

function [spks,tWin] = spk_getSpikes(s,tWin,Ev)

DESCRIPTION 

 get the spikes within a time window
 spks = spk_getSpikes(s,tWin,Ev)

 tWin ........... relative event window
 Ev ............. char/numeric; if set tWin works as an offset to the event

CROSS-REFERENCE INFORMATION

This function calls:
  • spk_SpikeChanNum returns channel data
  • spk_TrialNum returns and check for number of trials in @al_spk object
  • spk_getEventWindow returns start and end time of a window defined by an event(s)
This function is called by:
  • spk_ChronuxGetSpike extract spike data to chronux format: struct(chan/trials).times; in SEC.!
